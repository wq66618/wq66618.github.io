1. GitHub创建个人仓库

2. 获得个人网站域名

3. GitHub创建个人仓库

4. 安装Git

git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub注册邮箱"
ssh-keygen -t rsa -C "你的GitHub注册邮箱"

绑定github –> setting –> sshkey

5. 安装Node.js

检测Node.js是否安装成功，在命令行中输入 node -v

6. 安装Hexo

在电脑常里创建一个文件夹，可以命名为Blog，Hexo框架与以后你自己发布的网页都在这个文件夹中
使用npm命令安装Hexo，输入：

npm install -g hexo-cli

这个安装时间较长耐心等待，安装完成后，初始化我们的博客，输入：

hexo init blog

注意，这里的命令都是作用在刚刚创建的Blog文件夹中。

为了检测我们的网站雏形，分别按顺序输入以下三条命令：

hexo new test

hexo g

hexo s

三个命令依次是新建一篇博客文章、生成网页、在本地预览的操作。

这些命令在后面作介绍，完成后，打开浏览器输入地址：

localhost:4000

命令

npm install hexo -g #安装Hexo
npm update hexo -g #升级
hexo init #初始化博客
hexo n "我的博客" == hexo new "我的博客" #新建文章
hexo g == hexo generate #生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy #部署
hexo server #Hexo会监视文件变动并自动更新，无须重启服务器
hexo server -s #静态模式
hexo server -p 5000 #更改端口
hexo server -i 192.168.1.1 #自定义 IP
hexo clean #清除缓存，若是网页正常情况下可以忽略这条命令

7. 推送网站

配置文件_config.yml，翻到最后修改为：

deploy:
type: git
repo:在GitHub上创建仓库的完整路径
branch: master

保存站点配置文件。

npm install hexo-deployer-git --save
hexo clean 
hexo g 
hexo d

地址栏输入你的放置个人网站的仓库路径

8. 绑定域名

第一步:ping你的 http://xxxx.github.io 的ip地址，为A解析记录。记录类型是CNAME，CNAME的记录值是：你的用户名.http://github.io
第二步:登录GitHub，进入之前创建的仓库，点击settings，设置Custom domain，输入你的域名
点击save保存。
第三步，进入博客文件夹 ，进入source目录下，创建一个记事本文件，输入你的域名,另存为所有文件格式
完成这三步，进入blog目录中，按住shift键右击打开命令行，依次输入：

hexo clean
hexo g
hexo d

9. 更换主题

git clone https://主题地址 themes/xxx

_config.yml配置文件 –> Scheme Settings –> themes

10. 发布文章

hexo n “博客名字”

hexo new “博客名字”

source文件夹中的_post文件夹中多了一个 博客名字.md 文件，使用Markdown编辑器打开
